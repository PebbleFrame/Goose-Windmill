This app is a project by several students at Hack Reactor. It is by no means perfect, so suggestions for improvement are definitely welcome!

The "score" is based on the official Hacker News ranking algorithm, but I wasn't able to figure out how to implement it exactly, so right now it's just a loose approximation.

The original developers of the web version of this app are Kenny Tran, Matthew Rourke, Jennifer Bland, and Sid Sukumar. I, Michael Cheng, designed this mobile version for a legacy project, working in a group with Christina Holland, Jeff Peoples, and Vinaya Gopisetti.
